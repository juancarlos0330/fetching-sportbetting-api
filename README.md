# 👷‍♂️🚧 Fetching Betting API 🚧👷‍♂️

## Getting started

### Prerequisites

Clone this repo.

```shell
git clone https://github.com/juancarlos0330/fetching-sportbetting-api.git
```

### Installation

install dependencies

```shell
npm install
```

### Usage

To run locally for development

```shell
npm start
```

To build a bundled transpiled javascript app

```shell
npm run build
```
